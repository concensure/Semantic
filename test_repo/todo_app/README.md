# Todo App Demo (Semantic A/B)

This demo project is used by the semantic layer end-to-end A/B benchmark suite.

## Location

- Source: `test_repo/todo_app/src/`
- Task suite: `test_repo/todo_app/ab_test_suite_tasks.json`

## Covered Behaviors

- due date creation and updates
- deterministic priority ordering
- tag add/remove/filter flows
- overdue task detection
- UI menu integration task

## Notes

- This demo is intentionally small so task-level context and retrieval behavior are easy to inspect.
- No secrets should be stored in this folder.
